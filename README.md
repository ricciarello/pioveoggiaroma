# 🌧️ Piove Oggi a Roma?

> La risposta definitiva alla domanda esistenziale che ogni romano si pone prima di uscire di casa.

Hai in programma un weekend fuori? Una passeggiata al centro? O semplicemente devi prendere la moto e vuoi evitare di trasformarti in un sub? 🏍️

**Piove Oggi a Roma** azzera il rumore di fondo dei classici siti meteo (pieni di grafici complessi, percentuali inutili e pubblicità) e ti sputa in faccia un'unica, brutale verità in tempo reale: **Piove o non piove?**

Puoi visitarlo qui 👉🏻 [Piove Oggi a Roma🌧️](https://ricciarello.github.io/pioveoggiaroma/)

---

## 🚀 Funzionalità Chiave

* **Risposta Binaria Immediata:** Nessuna interpretazione richiesta. Sì o No. E anche informazione su domani!
* **UX/UI:** Palette scura ad alto contrasto per una consultazione rapidissima anche sotto la luce del sole (o mentre sei al semaforo).
* **Dati Real-Time:** Integrazione diretta con API meteo live per riflettere la situazione esatta della Capitale.
* **Mobile First:** Ottimizzato specificamente per essere consultato al volo dallo smartphone.

---

## 🛠️ Tech Stack

Il progetto è strutturato seguendo l'approccio *KISS (Keep It Simple, Stupid)* per garantire performance e zero costi di gestione:

* **Data Source:** OpenWeatherMap API / Weather API (Live data tracking).
* **Frontend:** `HTML5` / `CSS3` con font custom a tema cyberpunk e layout ultra-responsivo.
* **Deployment:** `GitHub Pages` per un hosting statico e rapido.
